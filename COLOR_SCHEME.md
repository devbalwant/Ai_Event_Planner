# EventPro Color Scheme

## Primary Colors

- **Maroon Primary**: `#8B1538` - Main brand color, navbar, hero sections
- **Maroon Dark**: `#5A0E24` - Dark backgrounds, footer, dark mode primary
- **Maroon Light**: `#A61D45` - Hover states

## Secondary Colors

- **Cream**: `#E8DCC4` - Light backgrounds, text on dark
- **Cream Dark**: `#D4C8B0` - Secondary text on dark backgrounds

## Accent Colors

- **Gold**: `#C4A574` - Buttons, accents, highlights
- **Gold Dark**: `#B09560` - Button hover states
- **Gold Light**: `#D4B584` - Subtle accents

## Usage Guide

### Backgrounds

- Light mode main: `bg-[#E8DCC4]`
- Dark mode main: `bg-[#5A0E24]`
- Light mode cards: `bg-white`
- Dark mode cards: `bg-[#8B1538]`

### Text

- Light mode primary: `text-[#8B1538]`
- Dark mode primary: `text-[#E8DCC4]`
- Light mode secondary: `text-[#8B1538]/70`
- Dark mode secondary: `text-[#D4C8B0]`

### Buttons

- Primary: `bg-[#C4A574] hover:bg-[#B09560] text-[#5A0E24]`
- Secondary: `border-[#E8DCC4] hover:bg-[#E8DCC4] text-[#E8DCC4] hover:text-[#8B1538]`

### Borders

- Light: `border-[#C4A574]/30`
- Dark: `border-[#C4A574]/20`

### Role Badges

- Admin: `bg-[#5A0E24]`
- Organizer: `bg-[#C4A574]`
- Attendee: `bg-[#8B1538]`
